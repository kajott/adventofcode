# [2020, Day 1: Report Repair](https://adventofcode.com/2020/day/1)

The input consists of (typically) 200 4-digit numbers.

**Part 1** asks to find two numbers that sum to 2020.

**Part 2** asks to find _three_ numbers that sum to 2020.


## Solution Notes

Relatively complex for a starter puzzle, as it requires two or three nested loops.

* Part 1, Python: 79 bytes, <100 ms
* Part 2, Python: 98 bytes, ~600 ms
